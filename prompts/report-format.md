# Prompt: Format the Daily Report

After summarizing each blogger individually, assemble the final report using this template.

## Report structure

```
# 📡 Blogger Radar — {date} · {weekday}

Tracking {N} creator(s) · Generated by Blogger Radar

---

## {Blogger Name}

*Tags: tag1 · tag2* | *Active on: platform1, platform2* | *Posts analyzed: N*

> {blogger.note if present}

**🔥 Highlights**
- {highlight 1}
- {highlight 2}
...

**🛠️ New Tools / Products**: tool1, tool2

**📊 Content Strategy**
- {observation 1}
- {observation 2}

**💬 Engagement Signals**: {one sentence}

**📈 Trend**: ↑ Rising | → Stable | ↓ Declining

**💡 Content Opportunities**
- {opportunity 1}
- {opportunity 2}

---

(repeat for each blogger)

---
*Generated by blogger-radar · {date} UTC · [source](https://github.com/topics/blogger-radar)*
```

## Formatting rules

1. **Order bloggers** by `trend_direction`: rising first, then stable, then declining.
   Within the same trend, order alphabetically by name.

2. **Omit empty sections**: if a blogger has no `new_products_or_tools`, skip that line entirely.
   Never write "None" or "N/A".

3. **Link every highlight** that references a specific piece of content:
   `- Published a deep-dive on RAG retrieval → [link](url)`

4. **Keep it scannable**: use bullet points, not paragraphs. Each bullet max 2 lines.

5. **Report header** should state the date and total blogger count.

6. **Footer** is always a single italic line with generation timestamp.

## For bilingual reports (language: bilingual)

Interleave English and Chinese paragraph by paragraph within each blogger section:

```
**🔥 Highlights**
- [English highlight 1]
- [English highlight 2]

**🔥 本周亮点**
- [Chinese translation of highlight 1]
- [Chinese translation of highlight 2]
```

Do NOT output all English sections first then all Chinese sections.
Interleave them section by section for each blogger.

## For Chinese-only reports (language: zh)

Produce the entire report in Chinese, using these section labels:
- 🔥 本周亮点
- 🛠️ 新工具 / 产品
- 📊 内容策略观察
- 💬 互动信号
- 📈 趋势判断
- 💡 可借鉴选题方向
